Authentication and Autorization

All API calls must use either simple or authorized access. Many API methods required authorized access, but some can use either.

Simple access

  • Doesn't access any private user data.
  • Needs an API key.

Authorized access

  • Accesses private user data.
  • Before calling them, user must grant your app access to their data.

Using some of this APIs require authorization.

Requirements

  • Credentials
  • Access Token
  • Use the Token for Authorization
