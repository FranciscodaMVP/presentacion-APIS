GoogleTranslator

With Google Translate API, you can dynamically translate text between thousands of language pairs.

The Google Translate API lets websites and programs integrate with Google Translate API programmatically.

It provides the collection language() that includes the methods:

  • detections(): Detect the language of text.
  • language(): List the source/target languages supported by the API.
  • translation(): Returns text translations from one language to another.
